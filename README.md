# Course Catalog Website - LaunchED Global

A sleek, mobile-responsive EdTech catalog built with semantic HTML5, CSS3 (Flexbox/Grid), and Vanilla JavaScript. This project showcases a real-world client-facing interface with dynamic tab switching and course enrollment features.

##  Live Demo
[View Live Project](https://passayah45-lgtm.github.io/Launched_global_courses_catalog/) 
*(Note: Ensure you have enabled GitHub Pages in your repo settings for this link to work)*

##  Features
- **Dynamic Course Grid**: Renders courses from a JSON-like data structure within JavaScript.
- **Tab Navigation**: Seamlessly switch between **Courses**, **Mentors**, and **Schedules** using a single-page logic.
- **Interactive Enrollment**: Features a simulated enrollment system that updates UI state instantly using a JavaScript `Set`.
- **Dark Mode UI**: Professional EdTech aesthetic using CSS variables, modern gradients, and a mobile-first approach.

## 🛠️ Built With
- **HTML5**: Semantic structure for better accessibility.
- **CSS3**: Custom properties (variables), Flexbox, and Grid for a responsive layout.
- **JavaScript (ES6)**: DOM manipulation, event handling, and state management.
